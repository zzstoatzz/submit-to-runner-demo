## using `submit_to_runner` for concurrent subflows w/ `prefect`

#### run the example
```bash
git clone https://github.com/zzstoatzz/submit-to-runner-demo.git
cd submit-to-runner-demo
docker compose up
```
[navigate to localhost:4200](http://localhost:4200)
